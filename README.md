# IT Quest: Мисия Академия

## 1. Име на проекта

**IT Quest: Мисия Академия**

## 2. Кратко описание

Това е локална уеб игра за училищни събития, отворени врати, изложби и кариерни форуми. Учениците въвеждат име или псевдоним, преминават през 5 кратки IT мисии, печелят точки, получават титла и се записват в класация.

## 3. Цел на играта

Играта има две основни цели:

- Да забавлява и ангажира ученици от приблизително 4. до 7. клас.
- Да представи по приятен начин теми като пароли, фишинг, компютърни мрежи, логика и алгоритми.

## 4. Използвани технологии

- **Front-end:** HTML, CSS, JavaScript
- **Back-end:** Node.js + Express
- **База данни:** SQLite
- **Звук:** Web Audio API

## 6. Инсталация

Уверете се, че имате инсталиран Node.js.

```bash
npm install
```

Ако сте на Windows PowerShell и стандартната команда `npm` е блокирана от локални политики, използвайте:

```bash
npm.cmd install
```

## 7. Стартиране

```bash
npm start
```

или:

```bash
npm.cmd start
```

След стартиране отворете:

```text
http://localhost:3000
```

## 8. Как се играе

1. На началния екран въведете име или псевдоним.
2. Натиснете **„Старт на мисията“**.
3. Отговорете на 5 кратки мисии.
4. След финала ще видите точки, време, бонус и титла.
5. Резултатът се записва автоматично в класацията.

## 9. Как работи точкуването

- Верен отговор: **+100 точки**
- Невален отговор, но участие: **+20 точки**

Максимален базов резултат: **500 точки**

Бонус за време:

- Под 45 секунди: **+50 точки**
- До 70 секунди: **+30 точки**
- До 90 секунди: **+10 точки**
- Над 90 секунди: **+0 точки**

Максимален възможен резултат: **550 точки**

Титли:

- 0–150 точки: **Начинаещ изследовател**
- 151–300 точки: **Дигитален помощник**
- 301–450 точки: **Млад кибергерой**
- 451–550 точки: **Бъдещ IT специалист**

## 10. Как работи класацията

Класацията показва:

- място
- име
- точки
- време
- титла
- дата и час

Подредбата е:

1. По най-висок резултат
2. При равен резултат по най-кратко време
3. При пълно равенство по най-нова дата

Първите трима участници са оцветени специално като злато, сребро и бронз.

## 11. Как се изчиства класацията

1. Отворете екрана **„Класация“**.
2. Натиснете дискретния бутон **„Админ“** в долната част.
3. Въведете администраторския код.
4. Потвърдете изчистването.

По подразбиране кодът е:

```text
academy2026
```

## 12. Как се сменя администраторският код

Има два начина:

### Вариант 1: През променлива на средата

Windows PowerShell:

```powershell
$env:ADMIN_CODE="nov-kod-тук"
npm start
```

### Вариант 2: Директно в кода

Отворете [server.js](./server.js) и сменете стойността на:

```js
const ADMIN_CODE = process.env.ADMIN_CODE || "academy2026";
```

## 13. Как се променят въпросите

Въпросите се намират в [public/app.js](./public/app.js) в масива `missions`.

Всеки обект описва една мисия:

- `title`
- `question`
- `options`
- `correctAnswer`
- `correctExplanation`
- `wrongExplanation`
- `type`

За мисията с подреждане се използват:

- `items`
- `correctOrder`

След редакция запазете файла и рестартирайте сървъра.

## 14. Как се използва играта на училищно събитие

- Инсталирайте проекта предварително на лаптопа.
- Проверете локално дали `npm start` отваря играта правилно.
- Отворете страницата на цял екран.
- По желание включете **„Режим за събитие“** от самото приложение.
- Оставете класацията достъпна, за да могат участниците да виждат резултатите си веднага.

Препоръчително е:

- да има мишка или тъчпад с лесен достъп
- да се използва външно захранване
- да се направи тест с няколко поредни игри преди началото на събитието
